# Punto 2

Una función Lamda de Python, con un gramática regular que comprenda el funcionamiento de la misma. 

## FUNCIONALIDADES 

- Debe devolver un ACEPTA o un NOACEPTA.

## REQUISITOS 

- **LEX** Ultima versión.
- **Virtualbox** ultima versión.
- **Kali linux**ultima versión.
- **PYTHON** versión dada en la terminal por defecto.

## USO

1. **EJECUCIÓN**:
   - Crea el archivo .txt.
   - Crea el archivo lambda.l
   - En cada archivo tenemos el código ejecutable. 
3. **INTERACCIÓN**:
   - ACEPTA o UN NOACEPTA dependiendo.
4. **RESULTADOS**:
   - ACEPTA.
   - NOACEPTA.
