# Punto 1 

Este es un programa en AWK que acepta las siguientes expresiones regulares:
+
++
[0-9]+
([0-9]+)”.”([0-9])+


## FUNCIONALIDADES 

- EN cada expresión regular debe devolver el token.

## REQUISITOS 

- **AWK** Ultima versión.
- **Virtualbox** ultima versión.
-**Kali linux**ultima versión.

## USO

1. **PREPARACIÓN**:
   - Tener instalado AWK en Kali linux.
   - Tener el codigo preparado para la ingresar en la terminal.
2. **EJECUCIÓN**:
   - Crea el archivo main, archivo guardado main.
   - Codigo en awk con la gramatica, para asi tener nuestros archivos.
   - Crea nano token.awk para que nos devuelva el token.
3. **INTERACCIÓN**:
   - Ingresa nos debe devolver un token.
4. **RESULTADOS**:
./token.awk: 1: +
./token.awk: 2: ++
./token.awk: 3: [0-9]+
./token.awk: 4: hello
